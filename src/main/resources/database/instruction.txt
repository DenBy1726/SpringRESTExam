Файлы в данном каталоге служат для развертывания бд.

1)Удалить все таблицы созданные в базе данных для данного проекта
для этого запустить файл clear.sql
=====================
Если в бд создается новая таблица ее следует поместить в этот файл
=====================

2)Создать все таблицы, необходимые для данного проекта
для этого запустить файл make.sql
=====================
Если в бд создается новая таблица ее следует поместить в этот файл
Помещать только реально использующиеся бд, без тестовых. Только те которые в релиз. Которые будут нужны для работы
конечного продукта.
=====================

3)Заполнить таблицы данными, необходимыми для минимальной работы, тестирования.
для этого запустить файл fill.sql
